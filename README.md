# Tree Risk Assessment Calculator

A free, offline-capable mobile web app for arborists and tree risk assessors. Implements the two-stage risk matrix from the ISA Best Management Practices for tree risk assessment.

## Features

- **Three-input risk calculator** — Likelihood of Failure, Likelihood of Impacting Target, and Consequences of Failure
- **Occupancy Rate selector** — dynamically enables/disables valid Likelihood of Impact options per TRAQ methodology
- **Live risk rating** — color-coded result updates instantly as selections are made (Low / Moderate / High / Extreme)
- **Risk matrix reference** — long-press the result bar to view both underlying matrices with your active selections highlighted
- **Definition pages** — long-press any rating button for its definition and examples (content placeholders ready to fill)
- **Offline capable** — runs entirely in the browser with no internet connection required after first load
- **Mobile-first** — designed for one-handed use in the field on Android and iOS

## Usage

Open `index.html` in any modern mobile browser, or visit the GitHub Pages URL. No installation or internet connection required.

To install as a home screen app on Android: open in Chrome → tap the menu → "Add to Home Screen".
